# OpsHelper AI 🤖

Assistente virtual inteligente desenvolvido com Inteligência Artificial para auxiliar estudantes e profissionais iniciantes em dúvidas relacionadas à infraestrutura e DevOps.

## 📌 Sobre o projeto

O OpsHelper AI é um assistente virtual criado com o objetivo de facilitar o aprendizado e a resolução de problemas comuns na área de tecnologia.

Através de uma base de conhecimento organizada, o assistente consegue interpretar dúvidas relacionadas a infraestrutura, apresentar explicações simples e orientar o usuário sobre possíveis próximos passos.

O projeto foi desenvolvido como parte do desafio **"Construa Seu Assistente Virtual Com Inteligência Artificial"**, aplicando conceitos de:

* Engenharia de Prompts;
* Organização de Base de Conhecimento;
* Desenvolvimento em Python;
* Aplicação prática de Inteligência Artificial.

---

## 🎯 Objetivo

Criar um assistente capaz de auxiliar usuários em tarefas relacionadas a:

* Linux;
* Docker;
* Git e GitHub;
* Redes;
* SSH;
* AWS e infraestrutura em nuvem.

O assistente busca fornecer respostas claras e confiáveis, evitando informações inventadas e indicando quando não possui conhecimento suficiente sobre determinado assunto.

---

## 👥 Público-alvo

O OpsHelper AI foi pensado para:

* Estudantes de tecnologia;
* Pessoas iniciando na área de infraestrutura;
* Analistas de suporte;
* Profissionais buscando consultas rápidas sobre comandos e conceitos técnicos.

---

## 🚀 Funcionalidades

Atualmente o projeto possui:

* Consulta em uma base de conhecimento própria;
* Respostas relacionadas a infraestrutura e DevOps;
* Organização de informações por categorias;
* Validação para evitar respostas fora da base cadastrada.
